Welcome to the Wrongful Dismissal Compliance Tool

This dashboard helps Definitely-Not-Evil-Corp Inc. evaluate the legal risk of terminating an employee under Canadian Employment Law. Use the menu on the left to:

  • Create or update Employee Records (profile + evidence).
  • Run a New Termination Assessment by selecting an employee.

How to Use This Prototype (Marker Tutorial)

  1. Step 1 – Create an employee record: Click “Employee Records” and fill in the employee profile.
  2. Step 2 – Add evidence: Upload conceptual evidence files (not stored in MVP).
  3. Step 3 – Save the employee: Click “Save Employee Record.”
  4. Step 4 – Start an assessment: Go to “New Termination Assessment” and complete Steps 1–4.
  5. Step 5 – Run the AI-style risk check: Click “Run Risk Assessment.” Loading screen → Results page.
  6. Step 6 – Restart if needed: Use “Run New Assessment” on results page.

Note: All data is stored only in browser memory for this prototype and clears on refresh.
